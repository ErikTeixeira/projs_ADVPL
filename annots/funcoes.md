# Funções

## cValToChar

Função que **converte Numérico, Lógico e Data para Caracter**

```
    MsgInfo("Conversões:" + CRLF + ;
        "Data: "     + cValToChar(Date())   + CRLF + ;
        "Numérico: " + cValToChar(13.8)     + CRLF + ;
        "Lógico: "   + cValToChar(.T.)      + CRLF + ;
        "Caracter: " + cValToChar("Daniel") + CRLF, "Atenção")
```
